# LiuTech 快速部署指南

## 本地开发

### 构建和启动
```bash
# 构建项目
.\build.bat

# 启动服务
docker-compose up -d
```

### 访问地址
- Web前端: http://localhost:3000
- Admin后台: http://localhost:3001
- 后端API: http://localhost:8080
- AI服务: http://localhost:8081

## 服务器部署

### 1. 本地导出镜像
```bash
# 构建项目
.\build.bat

# 导出镜像（Windows）
.\镜像导出脚本.bat

# 导出镜像（Linux）
./镜像导出脚本.sh
```

### 2. 上传文件到服务器
上传到 `/opt/liutech/` 目录：
- `docker-images/` 目录下的所有 `.tar` 文件 → `images/`
- `sql.sql` 数据库初始化文件
- `ai_chat_tables.sql` AI数据库初始化文件（首次初始化使用）
- `服务器部署脚本.sh`

### 3. 服务器执行
```bash
cd /opt/liutech
chmod +x 服务器部署脚本.sh
./服务器部署脚本.sh
```

### 4. 访问地址

- 用户前端: http://服务器IP:80
- 管理后台: http://服务器IP:81
- 后端API: http://服务器IP:80/api
- AI服务: http://服务器IP:80/ai

## 常用命令
```bash
# 查看状态
docker compose ps

# 查看日志
docker compose logs -f

# 重启服务
docker compose restart
```
